# 두 번째 실습과제 : 크로마키 합성

## 기술 스택
**언어**: python  
**라이브러리**: OpenCV (cv2), NumPy

## 주요 기능 및 단계
**1. 이미지 전처리 :** 전경과 배경을 해상도를 동일하게 맞춥니다.  
**2. 색상 검출 :** cv2.inRange를  사용해 녹색 영역만 추출하는 마스크를 생성합니다.  
**3. 전경 추출 :** 마스크를 반전시켜 배경을 제외한 피사체만 떼어냅니다.  
**4. 배경 가공:** 배경이미지에서 피사체가 들어갈 자리를 검은색으로 비워둡니다.  
**5. 최종 합성:** cv2.add를 통해 합칩니다.
