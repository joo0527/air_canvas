# Air Canvas Project with OpenCV

OpenCV를 활용하여 웹캠 앞에서 빨간색 물체를 움직여 실시간으로 그림을 그릴 수 있는 **디지털 에어 캔버스**입니다.

## 주요 특징
- **실시간 좌표 추적**: HSV 색 공간 필터링을 통해 빨간색 객체의 중심점을 정확히 추적합니다.
- **고성능 그리기**: 매 프레임 좌표를 계산하는 대신 별도의 `Canvas` 이미지를 생성하여 `cv2.add` 연산으로 합성함으로써 리소스 소모를 최소화했습니다.
- **다이나믹 UI**: 현재 선택된 색상을 화면 왼쪽 상단에 실시간 텍스트로 표시합니다.

## 단축키 안내
| 키 | 기능 |
|:---:|---|
| **`r`** | 펜 색상을 **빨간색(Red)**으로 변경 |
| **`p`** | 펜 색상을 **랜덤**으로 변경 |
| **`c`** | 캔버스 초기화 (전체 지우기) |
| **`q`** | 프로그램 안전 종료 |

## 개발 환경
- **Language**: Python 3.11+
- **Library**: `opencv-python`, `numpy`
- **IDE**: VS Code
- **Environment**: venv (가상환경)
